# Input de dados

## O que é?

* Entrada de dados;

## Exemplo

* Como pegar dados:

    gets()
// ou    
    gets

* Sua inicialização:

    variavel = gets

* Apagar o \n(enter) do input após pegar um dado:

    variavel.chomp()

    // Pode salvar dentro de uma variavel

    outra_variavel = variavel.chomp()

    // Ou declara o chomp no get

    variavel = gets.chomp
